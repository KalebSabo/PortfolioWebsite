Home Lab

Introduction:
This is a Home Lab setup to run across my home network, IoT devices and virtual machines through AWS. With this setup I have a few ways to use this setup like backing up my home network devices or staging mock cyber attacks on one of my virtual machines.

Hardware:

  • Raspberry Pi 4 Model B/Ubuntu OS
  • Personal Desktop/Windows OS
  • Personal Laptop/Windows OS

Software:

  • AWS EC2 Instances
  • Wireshark
  • Cisco Packet Tracer
  • Apache
